# Конфигурация Devin для проекта Base44

Эта папка содержит конфигурационные файлы для агента Devin, оптимизированные для работы с проектом base44-app.

## Структура конфигурации

```
.devin/
├── config.json                 # Основная конфигурация проекта
├── AGENTS.md                   # Правила и инструкции для агента
├── vscode-recommendations.json # Рекомендации для VSCode/Windsurf
├── mcp-config.json            # Конфигурация MCP серверов
├── project-checklist.json     # Чеклист для разработки
└── README.md                  # Этот файл
```

## Описание файлов

### config.json
Основная конфигурация проекта, содержащая:
- Информацию о технологическом стеке
- Структуру путей и алиасов
- Доступные скрипты npm
- Настройки тестирования
- Docker конфигурацию
- Переменные окружения

### AGENTS.md
Подробная документация для агента, включающая:
- Общую информацию о проекте
- Технологический стек
- Структуру проекта
- Команды для разработки
- Правила кодирования
- Инструкции по работе с интеграциями
- Диагностику проблем

### vscode-recommendations.json
Рекомендации для настройки VSCode/Windsurf:
- Список рекомендуемых расширений
- Настройки редактора
- Конфигурация ESLint и Prettier
- Настройки Tailwind CSS

### mcp-config.json
Конфигурация MCP серверов для интеграций:
- Файловая система
- Git операции
- Память агента
- Playwright (опционально)
- Pinecone (опционально)

### project-checklist.json
Чеклисты для различных этапов работы:
- Настройка проекта
- Разработка
- Тестирование
- Деплой
- Обслуживание
- Устранение проблем

## Использование

Эти файлы автоматически используются агентом Devin для:
1. Понимания структуры проекта
2. Выполнения правильных команд
3. Следования правилам кодирования
4. Диагностики проблем
5. Эффективной работы с кодовой базой

## Обновление конфигурации

При изменениях в проекте обновляйте соответствующие файлы:
- Добавлены новые зависимости → обновите config.json
- Изменилась структура проекта → обновите config.json и AGENTS.md
- Новые скрипты → добавьте в config.json
- Изменились правила кодирования → обновите AGENTS.md
